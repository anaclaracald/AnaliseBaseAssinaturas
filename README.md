# 📊 Análise de Assinantes e Produtos - Região Sul

## Objetivo

Este projeto tem como objetivo analisar o comportamento de uma base de assinantes, identificando padrões de consumo, distribuição geográfica, perfil demográfico e nível de satisfação dos clientes.

A análise busca gerar insights estratégicos que possam apoiar a tomada de decisão em áreas como marketing, produto e operações.
